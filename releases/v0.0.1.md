# v0.0.1

The first release of **Lemonade Stand**, a desktop MCP client / agent harness
that drives local models served by Lemonade (`lemond`).

## Added
- Chat UI that streams the agent's thinking and tool activity.
- Agent tool-calling loop over MCP servers (stdio + Streamable HTTP transports).
- The Pantry: browse the Market and stock/unstock tools & skills.
- Model picker with Omni/agent-ready highlighting and runtime context sizing.
- Optional spoken replies (TTS).

## Requirements
- Requires a running `lemond` server: (https://github.com/lemonade-sdk/lemonade)
