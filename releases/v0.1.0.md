# v0.1.0 — Connect to your Lemonade server

_Release date: 2026-07-18_

This release makes a fresh install connect to Lemonade out of the box and lets
you configure the server connection from inside the app.

## Fixed
- **Packaged installs now find their bundled config.** The Pantry catalogue and
  configured MCP servers were empty on a fresh install because the app read
  `config/` from inside the read-only app bundle. Defaults are now seeded into a
  writable per-user directory on first run, so the Market is populated and your
  edits survive upgrades.
- **Correct default server port.** The app defaulted to `http://localhost:8000`,
  but Lemonade Server listens on **13305**. The default is now
  `http://localhost:13305/api/v1`, so the server-status indicator shows *online*
  without any configuration.

## Added
- **In-app connection settings.** Click the **server-status** pill in the top
  bar to set the Lemonade base URL and (optional) API key. The choice is saved
  and applied immediately — no `.env` edit or restart needed.

## Requirements
- Requires a running `lemond` server: (https://github.com/lemonade-sdk/lemonade)
