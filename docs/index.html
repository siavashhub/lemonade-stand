What it is

A local agent that turns your model into a doer.

Lemonade Stand wires a local language model to real tools over the Model Context Protocol. The main process runs the tool-calling loop and talks to lemond; the renderer streams the agent's thinking and every tool action into a live chat.

The agent can treat lemond's own /mcp gateway as just another MCP server, so chat, multimodal "omni", image generation and transcription tools show up right alongside third-party ones.

Safety by default

Nothing runs without your say-so.

Before any tool executes you get an approval card showing the exact server, tool, and arguments. Choose Allow once, Always allow for the session, or Deny, or flip on session-wide bypass when you're moving fast.

  • 🛡️
    Per-call approvalReview args before anything touches your machine.
  • ⚡
    Bypass toggleSession-scoped fast lane, cleared on restart.
  • 🔒
    Trusted by designServers run with the app's privileges, you gate them.

Pitchers

Scheduled tasks, poured fresh.

Pitchers run prompts on a timer or when the app opens, no babysitting. Each pour uses a per-task tool whitelist so scheduled runs can't be steered into unintended actions.
